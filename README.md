
# Curriculum

Com apenas um arquivo HTML, dá pra fazer muita coisa. Provei isso, elaborando o meu currículo.

## Estrutura

Os currículos são organizados por idioma e depois por finalidade:

```
pt-br/
  general/index.html                 Currículo geral (Software Engineer / Full Stack)
  roles/devsecops.html               Currículo focado em DevSecOps
  roles/especialista-em-ia.html      Currículo focado em IA (LLMs) - vaga CGE/CE
  companies/rt-data/rt-data.html     Currículo para vaga de Engenheiro de Dados - RT Data

en/
  general/index.html
  roles/devsecops.html
  roles/especialista-em-ia.html
  companies/rt-data/rt-data.html
```

- `general/`: currículo genérico, usado como base para novas variações.
- `roles/`: versões adaptadas para um tipo de vaga específico (não ligadas a uma empresa em particular).
- `companies/<empresa>/`: versões feitas sob medida para o processo seletivo de uma empresa específica. Cada nova aplicação ganha sua própria subpasta.
